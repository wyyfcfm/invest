# 多维模型｜芒格公司分析知识包使用说明

这套知识包的目标：把查理·芒格的多元思维模型，整理成一套可重复使用的公司分析系统，方便在“项目/多维模型”中被不同对话窗口共同引用。

## 推荐放置方式

1. 在 ChatGPT 的“多维模型”项目中，上传本知识包内的全部 `.md` 文件作为项目参考资料。
2. 打开 `00_项目总指令_可复制到项目说明.md`，把其中内容复制到项目的 Custom Instructions / 项目说明里。
3. 根据 `09_多窗口对话设计_标题与首条提示词.md`，在同一个项目中创建多个新对话窗口。
4. 每个窗口只负责一个任务，例如“模型库维护”“行业研究”“单家公司深度分析”“估值与安全边际”“复盘案例库”。
5. 分析任何公司时，优先让 AI 调用：
   - `01_公司分析总框架.md`
   - `02_芒格核心思维模型库.md`
   - 对应行业文件或模块
   - `04_公司分析卡片模板.md`
   - `07_风险反向清单.md`
   - `08_估值与下注框架.md`

## 使用原则

不要让 AI “套所有模型”。正确做法是：

> 先定义公司本质，再选择最相关的 5—8 个模型，找证据验证，再做反向推翻，最后形成估值与决策结论。

## 推荐的日常工作流

### 1. 初识一家公司

使用窗口：`04｜公司初筛与一句话定义`

目标：快速判断这家公司是否值得深入研究。

### 2. 深入研究一家公司

使用窗口：`05｜单家公司深度分析`

目标：形成完整公司分析卡片。

### 3. 专门拆行业

使用窗口：`03｜行业结构与关键变量研究`

目标：搞清楚行业利润来源、供需、竞争、周期和替代风险。

### 4. 专门看财务

使用窗口：`06｜财务质量与现金流分析`

目标：判断利润是不是真钱，增长是否创造价值。

### 5. 专门找风险

使用窗口：`08｜反向思维与毁灭清单`

目标：强制寻找反证，避免只讲好故事。

### 6. 最后做投资判断

使用窗口：`09｜估值、安全边际与下注决策`

目标：判断当前价格下是否值得买、等、放弃或继续跟踪。

## 每次和 AI 沟通时的万能开场

你可以直接复制：

```text
请使用“多维模型”项目中的公司分析框架、芒格核心思维模型库、风险反向清单和估值下注框架，帮我分析【公司名称/股票代码】。
请不要堆模型，只选择最关键的 5—8 个模型，并按以下结构输出：
1. 一句话定义公司
2. 赚钱公式
3. 行业结构
4. 核心护城河
5. 财务质量
6. 管理层与资本配置
7. 反向清单
8. 估值与安全边际
9. 跟踪指标
10. 最终结论
如果缺少最新数据，请先说明需要联网核验哪些信息。
```
