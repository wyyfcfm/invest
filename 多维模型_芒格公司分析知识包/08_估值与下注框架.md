# 估值与下注框架

好公司不等于好投资。估值的目的不是算出一个精确数字，而是判断：当前价格下，风险、收益、确定性和机会成本是否匹配。

## 1. 估值前先确认公司类型

不同公司不能用同一套估值方式。

| 公司类型 | 重点估值方法 | 核心风险 |
|---|---|---|
| 稳定消费品 | PE、DCF、股息率、自由现金流收益率 | 增长放缓、提价权减弱 |
| 高 ROIC 成长股 | DCF、PEG、长期现金流情景 | 增长不及预期、估值收缩 |
| 周期股 | PB、EV/EBITDA、周期均值盈利 | 在盈利高点误判便宜 |
| 资源股 | 资源价值、周期价格、现金成本 | 商品价格下跌、资本开支 |
| 金融股 | PB、ROE、股息率、资产质量 | 坏账、杠杆、尾部风险 |
| 平台公司 | DCF、用户价值、利润率成熟情景 | 监管、竞争、网络效应反转 |
| 公用事业 | 股息率、DCF、资产回报率 | 监管定价、资本开支、杠杆 |

## 2. 当前价格隐含预期

估值第一步不是问“值多少钱”，而是问：

> 当前市场价格到底隐含了什么假设？

检查问题：

1. 当前估值隐含未来收入增长多少？
2. 隐含利润率维持在什么水平？
3. 隐含 ROIC 是否长期高于资本成本？
4. 隐含终局市占率是否合理？
5. 隐含风险是否过低？

模板：

```text
当前价格隐含的核心预期是：
1. 收入未来【X】年保持【X%】增长；
2. 利润率维持在【X%】左右；
3. 公司护城河不会明显恶化；
4. 估值倍数维持在【X】倍附近。
如果这些假设偏乐观，则安全边际不足。
```

## 3. 三情景估值

不要只给一个估值，必须做悲观、中性、乐观三种情景。

| 情景 | 假设 | 估值方法 | 合理价值 | 潜在回报 | 概率 |
|---|---|---|---|---|---|
| 悲观 | 增长放缓、利润率下降、估值收缩 |  |  |  |  |
| 中性 | 正常增长、利润率稳定、估值正常 |  |  |  |  |
| 乐观 | 增长超预期、护城河增强、估值提升 |  |  |  |  |

结论模板：

```text
悲观情景下，可能亏损约【X%】。
中性情景下，可能收益约【X%】。
乐观情景下，可能收益约【X%】。
综合胜率、赔率和下行风险，我认为当前价格【有/没有】足够安全边际。
```

## 4. 安全边际

安全边际来自三个地方：

1. 买入价格明显低于保守价值。
2. 公司本身质量高，长期现金流确定性强。
3. 资产负债表稳健，最坏情况不容易永久损失本金。

检查问题：

- 悲观情景下是否仍能保住大部分本金？
- 是否只有乐观情景才能赚钱？
- 是否有现金流、资产或分红支撑下限？
- 估值是否处于历史低位或合理区间？
- 市场是否忽略了长期价值？

危险信号：

- 需要非常乐观的增长假设才合理。
- 当前利润处于周期高点。
- 公司质量一般，但估值不便宜。
- 杠杆高，下行风险不可控。

## 5. 概率赔率

投资判断不是简单判断“会涨还是会跌”，而是比较：

```text
期望值 = 悲观结果 × 悲观概率 + 中性结果 × 中性概率 + 乐观结果 × 乐观概率
```

实际使用时不用追求数学精确，重点是思考：

- 下行空间是否可承受？
- 上行空间是否足够补偿风险？
- 胜率是否来自证据，而不是感觉？
- 赔率是否被高估值压低？

## 6. 机会成本

每次买入前，都要问：

1. 这家公司是否明显优于我已经持有的公司？
2. 是否有更便宜、更确定、风险更小的机会？
3. 如果只能持有 5 年，我还愿意买吗？
4. 如果不能交易 3 年，我还敢买吗？
5. 它是否值得占用组合仓位？

结论模板：

```text
相对于其他可选机会，这家公司在【确定性/估值/成长/现金流/风险】方面的优势是【优势】；劣势是【劣势】。因此仓位上应当【重仓/中等/小仓/观察】。
```

## 7. 买入、等待、放弃、继续跟踪

最终结论必须转化为行动。

| 结论 | 适用情况 |
|---|---|
| 买入 | 公司质量高，估值有安全边际，关键风险可承受 |
| 等待 | 公司好，但价格不够便宜 |
| 放弃 | 公司质量、治理、财务或行业风险不可接受 |
| 继续跟踪 | 逻辑有吸引力，但关键数据不足或价格未到 |

## 8. 估值结论模板

```text
这家公司当前估值判断如下：

1. 公司类型：【消费/制造/平台/周期/金融/公用事业/其他】。
2. 适合的估值方法：【PE/PB/DCF/EV EBITDA/股息率/周期均值】。
3. 当前价格隐含预期：【简述】。
4. 悲观情景：合理价值约【】；潜在回报【】。
5. 中性情景：合理价值约【】；潜在回报【】。
6. 乐观情景：合理价值约【】；潜在回报【】。
7. 安全边际：【充足/一般/不足】。
8. 机会成本：【高/中/低】。
9. 行动结论：【买入/等待/放弃/继续跟踪】。
10. 触发条件：低于【价格/估值】重点关注；高于【价格/估值】风险收益不划算。
```
