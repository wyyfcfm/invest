# AI 沟通万能提示词合集

本文件用于在“多维模型”项目中快速调用不同分析模式。

## 1. 完整公司分析

```text
请使用“多维模型”项目中的公司分析框架、芒格核心思维模型库、行业模块选择器、公司分析卡片模板、风险反向清单和估值下注框架，帮我完整分析【公司名称/股票代码】。
请不要堆模型，只选择最关键的 5—8 个模型。
请按以下结构输出：
1. 结论先行
2. 一句话定义公司
3. 赚钱公式
4. 行业结构
5. 核心思维模型与证据
6. 护城河判断
7. 财务质量
8. 管理层与资本配置
9. 反向清单
10. 估值与安全边际
11. 跟踪指标
12. 最终结论
如果缺少最新数据，请先列出需要联网核验的信息。
```

## 2. 快速初筛

```text
请帮我快速初筛【公司名称/股票代码】，判断是否值得深入研究。
只输出：
1. 一句话定义
2. 赚钱逻辑
3. 可能的护城河
4. 最大风险
5. 需要验证的关键数据
6. 初筛结论：值得深入/继续观察/暂时放弃
```


## 4. 护城河分析

```text

7. 护城河强度结论
```

## 5. 财务质量分析

```text

```
