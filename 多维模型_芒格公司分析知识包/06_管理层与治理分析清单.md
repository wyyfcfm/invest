# 管理层与治理分析清单

管理层分析的核心不是听他说什么，而是看他长期怎么分配资本、怎么对待股东、怎么在压力下行动。

## 1. 股权结构

检查问题：

1. 控股股东是谁？
2. 实控人是否稳定？
3. 管理层是否持股？
4. 股权是否过度集中？
5. 中小股东权利是否容易被侵害？

证据：

- 年报股权结构
- 实控人变化
- 管理层持股
- 限售与减持记录

危险信号：

- 控股股东高比例质押。
- 实控人频繁变更。
- 管理层长期不持股但薪酬高。
- 大股东与上市公司之间关系复杂。

## 2. 激励机制

检查问题：

1. 管理层考核收入、利润、市值，还是 ROIC 和现金流？
2. 股权激励价格是否合理？
3. 行权条件是否过低？
4. 激励是否鼓励短期行为？
5. 员工、渠道、管理层、股东利益是否一致？

证据：

- 股权激励方案
- 薪酬结构
- 业绩考核指标
- 管理层持股变化

危险信号：

- 考核只重收入规模，不重利润质量。
- 低价股权激励过度稀释股东。
- 管理层在业绩高点大量减持。

## 3. 资本配置

检查问题：

1. 公司赚来的钱用于再投资、分红、回购、并购、降债，还是闲置？
2. 再投资回报率是否高？
3. 并购是否理性？
4. 回购是否在低估时发生？
5. 分红是否可持续？

证据：

- 历史资本开支
- 并购公告与后续商誉减值
- 分红率
- 回购价格
- 债务变化
- ROIC 趋势

危险信号：

- 主业现金流好，但热衷跨界并购。
- 高价并购后计提商誉减值。
- 高分红同时高负债。
- 公司低估时不回购，高估时回购做姿态。

## 4. 分红与回购

检查问题：

1. 公司是否有稳定分红政策？
2. 分红是否来自真实自由现金流？
3. 回购是否注销，还是用于股权激励？
4. 回购价格是否合理？
5. 公司是否在没有好投资机会时返还现金？

证据：

- 分红历史
- 回购公告
- 自由现金流
- 股本变化

危险信号：

- 分红靠借债维持。
- 回购后用于激励，没有减少股本。
- 管理层用回购配合减持。

## 5. 并购与扩张

检查问题：

1. 并购是否与主业相关？
2. 并购价格是否合理？
3. 并购后整合是否成功？
4. 商誉是否过高？
5. 跨界扩张是否超出能力圈？

证据：

- 并购历史
- 商誉规模
- 被并购资产业绩承诺完成率
- 后续减值
- 管理层解释

危险信号：

- 频繁跨界并购。
- 高商誉、高业绩承诺、高减值风险。
- 管理层用并购掩盖主业增长乏力。

## 6. 关联交易与代理问题

检查问题：

1. 是否存在大额关联交易？
2. 关联交易定价是否公允？
3. 是否有资金占用或对外担保？
4. 上市公司是否为集团输血？
5. 优质资产和利润是否留在上市公司？

证据：

- 关联交易公告
- 其他应收款
- 对外担保
- 资金占用说明
- 审计意见

危险信号：

- 其他应收款异常大。
- 对控股股东或关联方担保。
- 采购或销售高度依赖关联方。
- 审计意见带保留或强调事项。

## 7. 声誉与承诺兑现

检查问题：

1. 管理层过去承诺是否兑现？
2. 信息披露是否透明？
3. 危机时是否保护股东和客户？
4. 是否尊重长期声誉？
5. 是否存在财务造假、监管处罚或重大失信？

证据：

- 历年年报管理层讨论
- 业绩预告准确性
- 监管处罚
- 投资者交流记录
- 媒体和诉讼信息

危险信号：

- 频繁改口。
- 重大事项披露不及时。
- 多次业绩变脸。
- 曾经损害股东、客户或供应商利益。

## 8. 管理层结论模板

```text
这家公司管理层整体判断为【优秀/合格/存疑/较差】。
主要优点是【优点1、优点2】。
主要风险是【风险1、风险2】。
从历史资本配置看，管理层更倾向于【再投资/分红/回购/并购/扩张/保守经营】。
我最需要继续跟踪的是【指标或行为】。
```
