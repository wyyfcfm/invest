# 风险反向清单｜毁灭公司与毁灭投资逻辑

芒格式公司分析的重点之一：先避免大错。每次分析公司，都要写一页反向清单。

## 1. 核心反向问题

必须回答：

1. 什么会让这家公司利润大幅下降？
2. 什么会让护城河失效？
3. 什么会导致现金流断裂？
4. 什么会让管理层毁灭价值？
5. 什么会让当前估值变得危险？
6. 我最可能看错哪里？
7. 哪些指标恶化说明投资逻辑被破坏？

## 2. 行业风险

检查问题：

- 行业需求是否可能下降？
- 人口、技术、政策、消费习惯是否在变化？
- 行业增长是否已经过了高峰？
- 需求是否被提前透支？
- 行业是否受宏观周期强烈影响？

触发指标：

- 行业销量连续下滑。
- 价格持续下降。
- 库存持续上升。
- 订单减少。
- 下游客户资本开支收缩。

反向结论模板：

```text
如果行业需求从【增长/稳定】变为【停滞/下滑】，且公司无法通过提价、市占率提升或成本下降抵消，则原有增长逻辑需要重新评估。
```

## 3. 供给与竞争风险

检查问题：

- 行业是否容易扩产？
- 高利润是否吸引新进入者？
- 是否存在价格战？
- 产品是否同质化？
- 竞争对手是否有更低成本或更强渠道？

触发指标：

- 行业资本开支大幅上升。
- 在建产能集中释放。
- 毛利率持续下降。
- 公司被迫降价。
- 市占率提升但利润率下降。

反向结论模板：

```text
如果新增供给释放导致行业价格持续低于成本曲线中位数，公司即使是龙头也可能面临利润均值回归。
```

## 4. 替代风险

检查问题：

- 是否存在更便宜、更方便、更高效的新方案？
- 客户真正需求是否发生变化？
- 新技术是否降低迁移成本？
- 公司是否只盯同行，而忽略跨行业替代？

触发指标：

- 新品类渗透率上升。
- 老产品销量下降。
- 年轻用户转移。
- 客户采购预算转向新方案。
- 公司研发方向落后。

反向结论模板：

```text
如果客户需求从【旧需求】迁移到【新需求】，而公司无法在新方案中保持优势，则原有护城河可能被绕开。
```

## 5. 护城河减弱风险

检查问题：

- 品牌是否失去年轻用户？
- 成本优势是否被同行追平？
- 网络效应是否反向运转？
- 转换成本是否下降？
- 渠道是否开始转向竞争对手？

触发指标：

- 毛利率低于同行。
- 用户留存率下降。
- 获客成本上升。
- 渠道库存上升。
- 复购率下降。
- 客户流失率上升。

反向结论模板：

```text
如果【关键护城河指标】连续恶化，说明公司优势可能正在变窄，不能再用过去的高估值或高利润率外推未来。
```

## 6. 财务风险

检查问题：

- 利润是否能变成现金？
- 应收账款是否恶化？
- 存货是否积压？
- 资本开支是否吞噬现金流？
- 负债是否过高？
- 债务是否集中到期？

触发指标：

- 经营现金流连续低于净利润。
- 自由现金流长期为负。
- 应收账款增速高于收入。
- 存货增速高于收入。
- 利息覆盖倍数下降。
- 短债压力上升。

反向结论模板：

```text
如果公司利润增长主要体现为应收和存货增长，而不是现金流增长，则利润质量需要下调。
```

## 7. 管理层风险

检查问题：

- 管理层是否盲目扩张？
- 是否进行高价并购？
- 是否有利益输送？
- 是否高位减持？
- 是否损害中小股东？

触发指标：

- 频繁跨界并购。
- 商誉大幅增加。
- 大股东质押比例上升。
- 关联交易上升。
- 分红不稳定。
- 管理层承诺反复落空。

反向结论模板：

```text
如果公司主业现金流被用于低回报扩张或利益输送，那么即使主业本身优秀，股东回报也可能很差。
```

## 8. 估值风险

检查问题：

- 当前价格隐含了什么增长预期？
- 市场是否已经把好消息反映完？
- 当前利润是否处于周期高点？
- 估值是否依赖乐观假设？
- 悲观情景下亏损是否可承受？

触发指标：

- 估值处于历史高位。
- 利润率处于历史高位。
- 市场叙事过度一致。
- 预期增长放缓但估值未调整。

反向结论模板：

```text
如果只有乐观情景才能获得合理回报，而悲观情景下下行空间很大，则安全边际不足。
```

## 9. 认知风险

检查问题：

- 我是不是只寻找支持买入的证据？
- 我是不是因为股价下跌就觉得便宜？
- 我是不是被知名投资人、市场共识或媒体叙事影响？
- 我是不是高估了自己对行业的理解？
- 我是不是忽略了自己不懂的关键变量？

常见误判：

- 确认偏误
- 锚定效应
- 从众心理
- 过度自信
- 损失厌恶
- 权威崇拜
- 叙事偏误

反向结论模板：

```text
我当前判断中最薄弱的环节是【薄弱环节】。在补充【数据/行业信息/财务验证】之前，不应把这个结论视为高确定性。
```
