# 财务质量检查清单

财务分析不是为了机械计算指标，而是为了验证商业判断：公司是否真正创造价值，利润是否能变成现金，增长是否会伤害股东。

## 1. 收入质量

检查问题：

1. 收入增长来自销量、价格、产品结构，还是并购？
2. 收入增长是否伴随应收账款异常增长？
3. 收入是否依赖少数大客户？
4. 收入是否存在明显季节性或周期性？
5. 收入是否靠渠道压货或提前确认？

关键指标：

- 收入增速
- 应收账款增速
- 合同负债/预收款
- 客户集中度
- 销量与价格拆分

危险信号：

- 收入增长明显快于现金流增长。
- 应收账款增长明显快于收入。
- 大客户收入占比过高且议价能力强。
- 渠道库存持续上升。

## 2. 毛利率质量

检查问题：

1. 毛利率高是因为品牌、成本、技术、渠道，还是周期涨价？
2. 毛利率是否高于同行？
3. 毛利率是否稳定？
4. 原材料涨价能否转嫁？
5. 毛利率是否处于历史异常高位？

关键指标：

- 毛利率趋势
- 同行毛利率比较
- 原材料价格
- 产品结构
- 单位成本

危险信号：

- 毛利率高但没有清晰护城河解释。
- 行业高景气时毛利率异常上升。
- 竞争加剧后毛利率快速下降。

## 3. 费用率质量

检查问题：

1. 销售费用是必要投入，还是为了维持虚假需求？
2. 研发费用是否形成长期竞争力？
3. 管理费用是否随规模扩张而摊薄？
4. 股权激励是否过度稀释股东？
5. 费用资本化是否激进？

关键指标：

- 销售费用率
- 管理费用率
- 研发费用率
- 费用率与收入规模关系
- 股权激励费用

危险信号：

- 收入增长依赖销售费用大幅增加。
- 研发费用高但没有产品成果。
- 规模扩大但管理费用率不降。

## 4. ROIC 与价值创造

检查问题：

1. ROIC 是否长期高于资本成本？
2. ROIC 是稳定的，还是周期高点？
3. 公司增长是否拉低 ROIC？
4. 高 ROE 是否来自高杠杆？
5. 留存收益是否创造了额外价值？

关键指标：

- ROIC
- ROE
- ROA
- 投入资本
- NOPAT
- 留存收益回报

危险信号：

- 收入增长但 ROIC 下降。
- ROE 很高但资产负债率也很高。
- 大量再投资却没有带来利润增长。

## 5. 现金流质量

检查问题：

1. 净利润能否转化为经营现金流？
2. 自由现金流是否长期为正？
3. 经营现金流是否依赖应付账款拖延？
4. 资本开支是维护性还是扩张性？
5. 公司是否有真实可分配现金？

关键指标：

- 经营现金流/净利润
- 自由现金流
- 资本开支
- 应收账款
- 存货
- 应付账款

危险信号：

- 净利润增长但经营现金流长期低于净利润。
- 自由现金流长期为负。
- 应收账款、存货大幅增加。
- 靠拉长账期改善现金流。

## 6. 营运资本

检查问题：

1. 应收账款是否恶化？
2. 存货是否积压？
3. 应付账款是否反映供应链强势，还是短期资金紧张？
4. 现金周转周期是否变长？
5. 公司增长是否大量占用营运资金？

关键指标：

- 应收账款周转天数
- 存货周转天数
- 应付账款周转天数
- 现金周转周期
- 坏账准备
- 存货跌价准备

危险信号：

- 应收账款周转变慢。
- 存货增长快于收入。
- 坏账准备不足。
- 存货跌价准备增加。

## 7. 资本开支

检查问题：

1. 公司维持现有竞争力需要多少资本开支？
2. 扩张资本开支回报率如何？
3. 折旧是否低估了真实维护成本？
4. 产能扩张是否会导致供给过剩？
5. 资本开支是否会侵蚀自由现金流？

关键指标：

- 资本开支/收入
- 资本开支/折旧
- 固定资产周转率
- 在建工程
- 产能利用率

危险信号：

- 大量在建工程即将投产但需求不足。
- 资本开支高企，自由现金流长期为负。
- 折旧政策过于乐观。

## 8. 负债与生存能力

检查问题：

1. 有息负债规模是否可控？
2. 利息覆盖倍数是否安全？
3. 债务是否集中到期？
4. 现金储备是否真实可用？
5. 行业低谷时是否还能还债？

关键指标：

- 资产负债率
- 有息负债/EBITDA
- 利息覆盖倍数
- 现金及等价物
- 短债/总债务
- 经营现金流/有息负债

危险信号：

- 短债比例高。
- 利息覆盖倍数下降。
- 现金很多但同时有大量受限资金。
- 融资环境依赖外部信用。

## 9. 财务质量总结模板

```text
这家公司的财务质量可以概括为：
1. 收入质量：【高/中/低】，原因是【证据】。
2. 利润质量：【高/中/低】，原因是【证据】。
3. 现金流质量：【高/中/低】，原因是【证据】。
4. 资本回报：【高/中/低】，ROIC 是否长期高于资本成本：【是/否/不确定】。
5. 负债风险：【高/中/低】，主要压力来自【债务/现金流/周期/资本开支】。
6. 最需要跟踪的财务指标是【指标1、指标2、指标3】。
```
