# K-000｜模型总表

| 编号 | 模型 | 核心用途 |
|---|---|---|
| K-001 | 安全边际 | 防止估值错误和永久损失本金 |
| K-002 | 能力圈 | 判断是否真正理解公司 |
| K-003 | 五力模型 | 分析行业利润结构 |
| K-004 | 价值链 | 找到利润创造与分配环节 |
| K-005 | 成本驱动因素 | 判断成本优势是否真实 |
| K-006 | 供需关系 | 判断价格、利润和周期位置 |
| K-007 | 规模经济 | 判断规模扩大是否增强竞争力 |
| K-008 | 网络效应 | 判断用户增长是否强化平台价值 |
| K-009 | 转换成本 | 判断客户黏性和迁移难度 |
| K-010 | 品牌与习惯 | 判断复购、提价权和心智优势 |
| K-011 | 低成本生产者 | 判断低谷和价格战中的生存力 |
| K-012 | 资本周期 | 判断高利润是否吸引扩产并压低回报 |
| K-013 | ROIC | 判断增长是否创造价值 |
| K-014 | 现金流 | 判断利润是否是真钱 |
| K-015 | 反向思维 | 找出毁灭公司或投资逻辑的路径 |
| K-016 | 概率赔率 | 比较胜率、赔率、下行和机会成本 |

后续可继续从 `多维模型_芒格公司分析知识包/02_芒格核心思维模型库.md` 拆分更多模型，例如反馈循环、路径依赖、替代效应、均值回归、复利、杠杆、激励机制、代理问题、资本配置、二阶效应、尾部风险和误判心理学。
