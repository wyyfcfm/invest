# 多维模型

本目录把书籍与原始知识包中的模型拆成可单独引用的研究工具。

使用原则：

1. 先定义公司和问题，再选择模型。
2. 每次公司分析只选最关键的 5-8 个模型。
3. 每个模型都必须回答：为什么适用、证据是什么、反向信号是什么、对结论有什么影响。
4. 不把模型当装饰词。模型必须落到财务、行业、竞争、管理层或用户行为证据上。

优先使用：

- 投资决策：`K-001_安全边际.md`、`K-002_能力圈.md`、`K-016_概率赔率.md`
- 行业结构：`K-003_五力模型.md`、`K-006_供需关系.md`、`K-012_资本周期.md`
- 护城河：`K-004_价值链.md`、`K-007_规模经济.md`、`K-008_网络效应.md`、`K-009_转换成本.md`
- 财务质量：`K-013_ROIC.md`、`K-014_现金流.md`
- 风险反证：`K-015_反向思维.md`
